### 步骤

- Fork 代码仓库并拉到本地
- 安装依赖

```
npm install
```

- 启动测试

```
npm run test
```

- 打开 `./test/test.spec.js` 修改代码跑通测试用例
- Pull Request
  - 怎么 Pull Request，参考：http://www.ruanyifeng.com/blog/2017/07/pull_request.html
